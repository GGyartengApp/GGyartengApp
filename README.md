### About Me

I'm a Computer Science student building **Peasah**, a farm management app for Ghanaian farmers — across both the mobile app (Expo / React Native) and the backend (Spring Boot, Java, PostgreSQL). I started as a mobile developer and am now learning backend development from scratch as part of the same project.

### Projects

**🌾 Peasah — Mobile App**
A farm management app for Ghanaian farmers, built with Expo and React Native. Features a full authentication system with session restoration, a home screen with live weather and commodity price data, animated dark/light theming, and haptic feedback.
`React Native` `Expo` `TypeScript`

**⚙️ Peasah — Backend**
The Spring Boot backend for Peasah, currently in development. Working through Java, Spring Data JPA, database migrations with Flyway, and Spring Security with JWT — learning backend development from the ground up while building a real production system.
`Spring Boot` `Java` `PostgreSQL` `Maven`

**⛪ Victory In Christ SDA Church App**
A mobile app built for a local SDA congregation under the Mid-West Ghana Conference. Includes a floating tab bar, a full Bible navigation flow, and a Sabbath School tab.
`React Native` `Expo` `TypeScript`
